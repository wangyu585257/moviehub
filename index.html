🚀 一键获取 API 密钥
选择任一平台注册，3 分钟即可开始创作。推荐从免费平台开始！
🟣 Replicate 🆓 新用户免费
  1. 点击下方按钮前往 Replicate 官网
  2. 使用 GitHub 账号一键注册（或邮箱注册）
  3. 登录后访问 API Tokens 页面
  4. 点击「Create token」生成密钥，复制 r8_xxxx... 格式的 Token
  5. 回到 MovieHub → 左下角「配置 API 密钥」→ 粘贴到 Replicate 栏
🔗 前往 Replicate 注册
🟡 SiliconFlow 🎁 注册送额度
  1. 点击下方按钮前往 SiliconFlow 云平台
  2. 手机号或邮箱注册，注册即送免费额度
  3. 登录后进入「控制台」→「API 密钥」
  4. 创建新密钥，复制 sk-xxxx... 格式的 Key
  5. 回到 MovieHub → 粘贴到 SiliconFlow 栏
🔗 前往 SiliconFlow 注册
🟢 HuggingFace 🆓 开源模型免费
  1. 点击下方按钮前往 HuggingFace
  2. 邮箱注册账号（支持 Google/GitHub 快捷登录）
  3. 登录后访问 Tokens 设置页
  4. 点击「New token」→ 权限选「Read」→ 复制 hf_xxxx...
  5. 回到 MovieHub → 粘贴到 HuggingFace 栏
🔗 前往 HuggingFace 注册
🔵 fal.ai 🎁 免费试用
  1. 点击下方按钮前往 fal.ai
  2. GitHub 账号一键登录
  3. 进入 Dashboard → Keys
  4. 创建 Key，复制 fal_xxxx...
  5. 回到 MovieHub → 粘贴到 fal.ai 栏
🔗 前往 fal.ai 注册
💡 密钥仅存储在你的浏览器本地，MovieHub 不会上传或保存你的任何密钥信息。
